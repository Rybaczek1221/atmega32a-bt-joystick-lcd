# Project Translation

## Polish title

Interaktywny sterownik mikrokontrolerowy z wyswietlaczem LCD, joystickiem i komunikacja Bluetooth.

## English title

Interactive microcontroller controller with LCD display, joystick and Bluetooth communication.

## English project description

This is a coursework project for the Digital Interfaces class built around an ATmega32A microcontroller. The system combines a 16x2 LCD, an analog joystick, Bluetooth communication and servo control.

The application offers a local menu controlled by the joystick and a remote text interface available through a Bluetooth terminal on a phone. It can display messages on the LCD, stream joystick readings, execute control commands and set servo positions.

## Main features in English

- menu navigation with an analog joystick,
- LCD message display,
- Bluetooth text mode,
- Bluetooth control mode with command parsing,
- joystick monitoring mode,
- servo test mode controlled by the joystick,
- UART responses sent back to the phone terminal.
